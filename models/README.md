# Models

## Overview

This folder stores the trained baseline model and the post-hoc calibrated models used in the static CTR calibration project.

The project follows a simple three-stage workflow:

1. train a raw logistic regression baseline on the training split
2. fit post-hoc calibration models on the validation split
3. evaluate all models on the held-out test split

## Files

### `lr_pipeline.joblib`
Raw logistic regression baseline pipeline.

This file stores the full `scikit-learn` pipeline, including:

- preprocessing for numerical features
- preprocessing for categorical features
- imputation
- scaling / one-hot encoding
- logistic regression classifier
